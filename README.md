# Premium Graduation Invitation

Mobile-first React + TypeScript + Vite invitation site for a graduation ceremony.

## Run Locally

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

## Customize

Edit all invitation content in:

```text
src/config/invitation.ts
```

Replace photos with optimized JPG files in:

```text
public/images/photo-1.jpg
public/images/photo-2.jpg
public/images/photo-3.jpg
public/images/photo-4.jpg
```

The project is ready for Vercel deployment with the default Vite settings.
